# 视频详情页重新设计规范

## 1. 设计概述

**布局风格**: 全屏海报 + 悬浮信息面板
**设计理念**: 信息丰富但不杂乱，视觉沉浸感强，操作直观

## 2. 页面布局

### 2.1 整体结构
```
┌─────────────────────────────────────────────────────┐
│                    全屏海报背景                       │
│                  (半透明渐变遮罩)                     │
│                                                     │
│  ┌─────────────┐              ┌──────────────────┐  │
│  │  右侧悬浮   │              │                  │  │
│  │  信息面板   │              │    主演/导演     │  │
│  │             │              │    横向滚动     │  │
│  │  - 标题    │              │                  │  │
│  │  - 评分    │              └──────────────────┘  │
│  │  - 简介    │                                    │
│  │  - 按钮    │                                   │
│  └─────────────┘                                    │
└─────────────────────────────────────────────────────┘
```

### 2.2 背景层
- 全屏海报/剧照作为背景
- 左侧 40% 应用从透明到深色 (#0d0d1a) 的渐变遮罩
- 底部 30% 应用从透明到主背景色的渐变遮罩
- 海报本身添加轻微模糊效果 (blur: 20px)

### 2.3 信息面板位置
- 垂直居中，左侧留白约 5%
- 宽度: 最大 500px
- 与底部渐变叠加，确保文字可读性

### 2.4 主演横向滚动区
- 位于信息面板下方
- 右侧固定"查看全部"箭头
- 卡片尺寸: 64x64 头像 + 2行文字

## 3. 详细组件设计

### 3.1 标题区域
```
星际穿越
Interstellar
```
- 主标题: 36px, 白色, 700 weight
- 英文副标题: 16px, #888, 位于主标题下方 4px
- 下方 12px 间距

### 3.2 元数据行
```
⭐ 8.7  │  2014  │  2h 49m  │  PG-13
```
- 评分星星: #f5a623
- 分隔符: 圆点 ● #666
- 字号: 14px, #ccc

### 3.3 类型标签
```
[科幻]  [冒险]  [剧情]
```
- 样式: 半透明紫色背景 + 紫色边框
- 颜色: #a5b4fc
- 间距: 8px
- 圆角: 16px (胶囊形)
- 可点击（点击后筛选同类型内容）

### 3.4 剧情简介
- 默认显示 3 行，超出部分截断并显示"展开"
- 展开按钮: "展开全部 ▼" / "收起 ▲"
- 字号: 14px, #ccc
- 行高: 1.7

### 3.5 操作按钮
```
┌────────────────┐  ┌─────┐  ┌─────┐
│   ▶ 开始播放   │  │ ♡  │  │ ↗  │
└────────────────┘  └─────┘  └─────┘
```
- **开始播放按钮**:
  - 尺寸: 高度 48px, 内边距 32px 水平
  - 样式: 渐变背景 (#6366f1 → #8b5cf6)
  - 文字: 16px, 白色, 600 weight
  - 圆角: 8px
  - Hover: 轻微上浮 + 阴影扩散

- **收藏按钮**:
  - 尺寸: 48x48, 圆形
  - 样式: 半透明背景 + 白边框
  - 图标: 空心心形 → 实心（已收藏）

- **分享按钮**:
  - 尺寸: 48x48, 圆形
  - 样式: 同收藏按钮
  - 图标: 分享箭头

### 3.6 剧集选择器（仅剧集显示）
```
第一季 ▼
┌──┐ ┌──┐ ┌──┐ ┌──┐ ┌──┐ ┌──┐
│1 │ │2 │ │3 │ │4 │ │5 │ │6 │
└──┘ └──┘ └──┘ └──┘ └──┘ └──┘
```
- 季度下拉选择器
- 集数网格: 6 列，行数自适应
- 当前选中集数: 紫色渐变背景
- 其他集数: 半透明背景

### 3.7 演职人员横向滚动
```
┌─────────────────────────────────────────────┐
│ 导演 & 演员                           查看全部 →
│ ┌─────┐ ┌─────┐ ┌─────┐ ┌─────┐ ┌─────┐   │
│ │头像 │ │头像 │ │头像 │ │头像 │ │头像 │   │
│ │名字 │ │名字 │ │名字 │ │名字 │ │名字 │   │
│ │角色 │ │角色 │ │角色 │ │角色 │ │角色 │   │
│ └─────┘ └─────┘ └─────┘ └─────┘ └─────┘   │
└─────────────────────────────────────────────┘
```
- 头像: 64x64, 圆形
- 名字: 13px, 白色
- 角色: 12px, #666
- 卡片宽度: 80px
- 间距: 16px

## 4. 交互状态

### 4.1 滚动行为
- 页面滚动时，信息面板固定在视口中央偏左
- 背景海报随滚动轻微视差移动

### 4.2 加载状态
- 骨架屏: 标题、标签、简介的灰色占位块
- 海报加载: 低质量缩略图 → 高质量原图渐显

### 4.3 错误状态
- 加载失败: 显示重试按钮
- 视频不存在: 跳转回首页

## 5. 响应式设计

### 5.1 大屏幕 (> 1200px)
- 信息面板宽度: 480px
- 左侧海报占: 40%

### 5.2 中等屏幕 (768px - 1200px)
- 信息面板宽度: 400px
- 背景海报全屏覆盖

### 5.3 小屏幕 (< 768px)
- 信息面板: 全宽，底部固定
- 背景海报: 固定高度 40vh，模糊处理
- 操作按钮: 底部固定栏

## 6. 技术实现要点

### 6.1 组件结构
```
VideoDetail.vue
├── BackdropBackground (海报背景)
├── InfoPanel
│   ├── TitleSection
│   ├── MetaRow
│   ├── GenreTags
│   ├── Synopsis (展开/收起)
│   ├── ActionButtons
│   └── EpisodeSelector (条件渲染)
├── CastScroll (演职人员横向滚动)
└── BackButton (返回)
```

### 6.2 样式规范
- 主色: #6366f1 (Indigo)
- 次色: #8b5cf6 (Purple)
- 强调色: #f5a623 (评分星星)
- 背景: #0d0d1a
- 文字主色: #fff
- 文字次色: #ccc
- 文字弱色: #666

### 6.3 动画规范
- 过渡时长: 200ms
- 缓动函数: ease-out
- 按钮 Hover: translateY(-2px) + scale(1.02)
- 展开收起: max-height 过渡
